
Welcome to the Cart Witnessing Availability Tracker! This tool is designed to make organizing, finding, and signing up for cart witnessing locations simple and efficient.

Below is a guide on how to use the application, split into instructions for general volunteers and system administrators.

👤 For Volunteers
The main dashboard shows all active cart witnessing locations. You do not need an account or password to sign up for a slot.

1. Signing Up for a Slot
Browse the available locations on the main screen.

Under the Volunteers section of a location, find an empty slot labeled Enter name....

Type your name and click the Join button.

Backup Slots: The last slot in every list is marked with an orange (BACKUP) tag. If the main slots are full, you can add your name here to step in if someone cancels.

2. Navigating to the Location
Click the Maps button on any location card.

This will instantly open Google Maps with the exact pin or Plus Code for that specific spot, allowing you to get directions easily.

3. Contacting Coordinators
Location Contact: If a location has an assigned overseer, you will see a green Contact [Name] button with a phone icon. Clicking this will instantly open a WhatsApp chat with that specific coordinator.

General Admin Help: At the very top right of the screen, click the Contact Admin button to send a WhatsApp message to the main system administrator for general help.

4. Locked Locations
If a location displays a red Locked badge, it means the administrator has temporarily closed this location, or all slots (including backups) are completely full. You will not be able to type your name into locked slots.
